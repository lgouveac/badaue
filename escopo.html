Checkout

Checkout simples com Stripe e gestão via CMS.

Upload de imagem

O time poderá subir a imagem do produto ou material diretamente no CMS.

Link de pagamento Stripe

O checkout usará link de pagamento do Stripe, sem desenvolvimento de fluxo transacional complexo.

Sem envio de e-mail

O projeto não inclui automação de e-mails nem disparos automáticos relacionados ao checkout.

Mais leveza operacional

A ideia é facilitar gestão e conversão, sem acoplar integrações desnecessárias nesta etapa.

Landing page builder

Criador de landing pages com modelos prontos.

O time escolhe um modelo e, a partir dele, consegue criar novas landings para campanhas, lançamentos, relatórios e ações comerciais sem depender de desenvolvimento a cada nova demanda.

Disclaimer

Itens fora do escopo desta proposta.

O projeto não inclui

  • Automações
  • Edições de imagem, exceto conversão para AVIF
  • Manutenção contínua
  • Edição de vídeo
  • SEO avançado
  • Integrações com sistemas externos

Após 60 dias da entrega, qualquer correção ou manutenção necessária deverá ser orçada separadamente.

O projeto inclui

  • Criação do novo site funcional
  • CMS para edição de conteúdo e imagens
  • Conversão técnica de imagens para AVIF
  • Implementação das tags de Analytics e Clarity
  • Estrutura de landing pages com modelos reutilizáveis

O escopo foi pensado para dar autonomia sem inflar a complexidade do projeto.

A proposta entrega uma base editável, comercialmente útil e preparada para expansão futura com menos retrabalho.
